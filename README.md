# Sliding Panel WordPress Plugin

## 📌 Overview
The **Sliding Panel** plugin is a lightweight and fully customizable WordPress plugin that adds a **sliding panel** to any WordPress website. It is designed to replace the **ClientEngage UberPanel** while improving performance, reducing plugin bloat, and offering admin customization options.

## 🎯 Features
✅ **Lightweight & Fast** – No unnecessary scripts or database queries.  
✅ **Customizable Content** – Admin can modify panel content from the WordPress settings.  
✅ **Easy Activation/Deactivation** – No need to modify theme files.  
✅ **Fully Responsive** – Works on desktops and mobile devices.  
✅ **Secure & Optimized** – Uses minimal CSS and JavaScript for better page speed.  
✅ **No External Dependencies** – Does not rely on third-party libraries.  

## 🚀 Installation
### **1. Download & Upload**
- Clone this repository or download the ZIP file.
- Upload the plugin folder `sliding-panel` to your `wp-content/plugins/` directory.

### **2. Activate the Plugin**
- Log in to your **WordPress Dashboard**.
- Navigate to **Plugins** > **Installed Plugins**.
- Find **Sliding Panel** and click **Activate**.

### **3. Configure the Panel**
- Go to **Settings > Sliding Panel**.
- Customize the panel content as needed.
- Save changes and check the panel on your website.

## 🛠️ How It Works
- A small **"☰ Open Panel"** button appears in the top right corner of the website.
- When clicked, the panel slides in from the right.
- The panel can be closed using the **"× Close"** button.
- Admins can modify panel content from the WordPress settings panel.

## 📂 File Structure
```
/sliding-panel/
│── sliding-panel.php  # Main plugin file
│── assets/
│   ├── panel-style.css  # CSS for panel
│   ├── panel-script.js  # JavaScript for functionality
│── includes/
│   ├── panel-content.php  # HTML structure for the panel
│   ├── panel-settings.php  # Admin settings page
```

## 🔧 Customization
### **Change the Panel Design**
Modify the `assets/panel-style.css` file to adjust colors, sizes, or positioning.

### **Modify Panel Content**
1. Go to **WordPress Dashboard** > **Settings** > **Sliding Panel**.
2. Edit the text inside the admin settings box.
3. Save changes.

### **Disable the Panel**
Deactivate the plugin from the **Plugins** page in the WordPress dashboard.
